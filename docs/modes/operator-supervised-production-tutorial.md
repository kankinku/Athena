# Athena Supervised Mode Production Tutorial

## Purpose

This tutorial explains how to validate Athena's supervised production mode end to end.

Use this when you want to verify that one operating mode of Athena is ready for bounded real use.

## Important Framing

This document is about a deployment mode, not Athena's product identity.

Athena is an autonomous research system.
Supervised mode is one way to operate that system safely while the broader autonomy stack continues to harden.

## What To Verify

- the validation suites pass
- the build passes
- the supervised soak artifact is generated
- the supervised checklist reflects real topology coverage
- the evidence document records real observed output
