# 성공 기준

이 문서는 Athena를 `자율 연구 시스템`으로 볼 때의 측정 가능한 성공 기준을 정의한다.

## SC-01: 목표 유지

Athena는 여러 반복을 거쳐도 현재 목표에서 벗어나지 않아야 한다.

## SC-02: 반복형 개선

Athena는 한 번 실행하고 끝나는 것이 아니라 `계획 -> 개선 -> 평가 -> 재설계`를 실제로 반복해야 한다.

## SC-03: 근거 기반 개선

개선안은 추측이 아니라 증거, 실험 결과, 비교 결과에 기반해야 한다.

## SC-04: 실행 가능성

Athena는 로컬 또는 원격 환경에서 실제 작업을 수행할 수 있어야 한다.

## SC-05: 평가 품질

Athena는 "돌아갔다"가 아니라 "개선되었는지"를 판단할 수 있어야 한다.

## SC-06: 운영 가능성

운영자는 현재 상태, 위험, 다음 액션, 실패 원인을 소스코드 없이 파악할 수 있어야 한다.

## SC-07: 자율성의 질

자율성은 감독 제거 수치가 아니라 `목표 유지 + 안전성 + 반복력`으로 평가한다.
