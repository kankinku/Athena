# 안전장치

이 문서는 Athena가 자율적으로 움직일 때 반드시 지켜야 하는 경계를 정의한다.

## 핵심 경계

- 허용된 경로 밖 수정 금지
- 위험한 명령과 민감 경로 보호
- 검증 없는 완료 선언 금지
- 롤백이 불가능한 고위험 작업의 무단 실행 금지
- 예산과 시간 한계 초과 시 자동 중단

## 안전장치의 역할

안전장치는 자율성을 막기 위한 것이 아니다. 자율성을 신뢰 가능한 범위 안에 가두기 위한 것이다.

## 현재 해석

Athena는 자율 연구 시스템이다. 따라서 guardrails는 "부가 기능"이 아니라 자율성을 가능하게 만드는 전제 조건이다.
