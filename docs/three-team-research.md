# Three-Team Research Workflow

Athena's staged research workflow supports the main autonomous improvement loop.

## The Three Roles

1. Collection
   - gather evidence, sources, and candidate ideas

2. Planning
   - turn evidence into ranked proposals and next actions

3. Simulation
   - execute experiments, bounded runs, or what-if validations

## Why It Exists

This is not a separate product inside Athena.

It is one way Athena organizes the same core loop:

```text
collect -> compare -> plan -> execute/evaluate -> keep, discard, or revisit
```

## Product Interpretation

If Athena is the autonomous research system, the three-team workflow is one runtime strategy for operating that system with clearer specialization and handoffs.
